# UI Standard

Renderer views should provide loading, status, and error states appropriate to the panel or route.

Current renderer theme direction:

- Use the Sakura Candy Pop palette for the Electron shell.
- Treat `sakuraPink` as the primary brand color.
- Use `mikuCyan` for focus rings, hover states, and interaction feedback.
- Keep text neutral on light surfaces for readability.
- Prefer gradients and soft tints over darker shades for depth.

Use this file to capture renderer-specific UI conventions as implementation expands.
